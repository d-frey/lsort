```
Usage: lsort [OPTION]... FILE
Sort an almost-sorted FILE, works in-place

  -c, --compare N            compare no more than N characters per line
  -d, --distance N           maximum allowed shift distance

  -v, --verbose              show progress and statistics
  -V, --version              print program version
  -?, --help                 give this help list

Mandatory or optional arguments to long options are also mandatory or optional
for any corresponding short options.

Report bugs to d.frey@gmx.de.
```
